## NextJs Crud
### This is a simple NextJS App for learning NextJS and SSG
## API Routes
<ul>
<li>/users -> Fetch All Users In Database</li>
<li>/users/id -> Fetch User By Id</li>
<li>/users/new [name, email, password] ->  Register New User</li>
</ul>
